# AGENTS.md

## About This Repo

Personal dotfiles managed with [chezmoi](https://www.chezmoi.io/). Supports macOS (work) and Bazzite Linux. Files are templated using chezmoi's Go templating to handle per-machine differences.

## Commit Style

Use [Conventional Commits](https://www.conventionalcommits.org/):

```
<type>: <short description>

Types: feat, fix, chore, refactor, docs
```

Examples:
- `feat: add starship prompt config`
- `fix: correct homebrew path on macOS`
- `chore: add ffmpeg to package lists`
- `docs: update README`
